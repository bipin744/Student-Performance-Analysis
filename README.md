# Student Performance Analysis using Python

## Project Overview

This project analyzes the relationship between:

- Family Income
- CGPA
- Python Experience
- Leadership Skills
- Expected Salary

using Python.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Business Questions Solved

- Total Unique Students
- Average CGPA
- Graduation Year Analysis
- Python Experience Analysis
- Family Income Analysis
- Salary Analysis
- Leadership Skills Analysis
- Event Analysis
